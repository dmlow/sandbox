Five Tools

For a weekend project, I decided to get some practice using the d3.js library by generating Nightingale charts for each of the 30 MLB teams according to their performance in each of the traditional "five tools" of baseball: hitting, hitting for power, fielding, throwing, and speed. The charts show, moving clockwise starting from the top, each team's strength in hitting (yellow); power (blue); fielding (green); throwing (red); and speed (navy).

While ultimately, I'd like be able to update the charts live as the season progresses, in this iteration, I'm using static data that I pulled from FanGraphs for the 2012 season.

I experimented with using a few different stats for each tool, before settling on the following:

  1. Contact rate for hitting,
  2. Isolated power for power,
  3. UZR/150 for fielding,
  4. Pitching WAR for throwing,
  5. FanGraph's Ultimate Base Running (UBR) for speed.

The one consistent trend here is that better "throwing" (in this case, pitching) teams tended to make the playoffs or at least finish higher in the rankings, while strenght in the four other tools didn't show as much consistency in results.
